--HR Analytics Dashboard (Power BI)<br>
This repository contains an HR Analytics Dashboard built entirely using Power BI. The goal of this project is to analyze key HR metrics and provide actionable insights related to employee attrition, workforce demographics, performance indicators, and organizational trends.<br>

--Tools & Technologies Used<br>
Power BI Desktop – Data modeling, DAX calculations, and dashboard creation<br>
Power Query – Data cleaning and transformation<br>
DAX (Data Analysis Expressions) – Measures and KPIs<br>
No coding languages were used. This project is 100% Power BI–based.<br>

--Key Features & Insights<br>
Employee Attrition Analysis<br>
Attrition by Department, Gender, and Job Role<br>
Average Age, Salary, and Years at Company<br>
Employee Count & Active Workforce Overview<br>
Job Satisfaction & Performance Trends<br>
Interactive filters and slicers for deep analysis<br>

--Repository Contents<br>
HR.csv- dataset from kaggle<br>
HR.pbix – Main Power BI dashboard file<br>
README.md – Project documentation<br>

--How to Use<br>
1. Download or clone this repository<br>
2. Open the .pbix file using Power BI Desktop<br>
3. Refresh data <br>
4. Use slicers and visuals to explore HR insights<br>
